iPhone 安装使用指南

📱 方法一：Safari 添加到主屏幕（推荐）

1

获取文件

将 index.html 文件通过「微信文件传输助手」或「AirDrop」发送到 iPhone

2

用 Safari 打开

在「文件」App 中找到 index.html，长按选择 「用 Safari 打开」（不能用微信内置浏览器打开）

3

点击分享按钮

点击 Safari 底部工具栏中间的 分享 📤 按钮

4

添加到主屏幕

在弹出菜单中向下滚动，找到 「添加到主屏幕」，点击后右上角点「添加」

5

像 App 一样使用 🎉

回到主屏幕，找到「英语知识库」图标，点击即可全屏运行，数据永久保存在手机本地

💡 重要提示：必须用 Safari 才能「添加到主屏幕」，Chrome、微信等浏览器不支持此功能。添加后从主屏幕打开，会隐藏地址栏，体验和原生 App 一样。

🌐 方法二：部署到网络访问（最方便）

将 index.html 上传到任意静态托管平台，生成网址，手机 Safari 打开后同样可以添加到主屏幕，随时随地访问。

推荐免费平台

• GitHub Pages：免费、稳定，github.com 注册后上传文件即可
• Vercel：vercel.com，拖拽 HTML 文件即可部署
• 腾讯云 COS：国内访问速度更快
• Netlify：netlify.com，拖拽文件夹直接发布

💾 数据说明

• 所有学习数据保存在 手机本地存储（localStorage），不需要网络、不需要账号
• 数据和 Safari / 主屏幕 App 绑定，不要清除 Safari 数据，否则记录会丢失
• 可以在「我的」页面点击 「导出学习数据」 备份 JSON 文件，换手机后导入即可恢复
